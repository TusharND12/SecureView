========================================
HOW TO DOWNLOAD (MANUAL)
========================================

Since automatic downloads are failing, download manually:

STEP 1: Download RetinaFace
----------------------------
1. Go to: https://github.com/opencv/opencv_zoo
2. Click on: models/face_detection_retinaface/
3. Find file: face_detection_retinaface_2023mar.onnx
4. Click "Download" or "Raw"
5. Save to: ~/.secureview/models/retinaface.onnx

OR use wget with correct URL (if you find it):
  cd ~/.secureview/models
  wget <correct-url> -O retinaface.onnx

STEP 2: Download SFace/ArcFace
-------------------------------
1. Go to: https://github.com/opencv/opencv_zoo
2. Click on: models/face_recognition_sface/
3. Find file: face_recognition_sface_2021dec.onnx
4. Click "Download" or "Raw"
5. Save to: ~/.secureview/models/arcface.onnx

OR use wget with correct URL (if you find it):
  cd ~/.secureview/models
  wget <correct-url> -O arcface.onnx

========================================
VERIFY INSTALLATION
========================================

After downloading, check:
  ls -lh ~/.secureview/models/

Should show:
  -rw-rw-r-- 1 user user 1.7M ... retinaface.onnx
  -rw-rw-r-- 1 user user  45M ... arcface.onnx

If files are 0 bytes or missing, download again!
